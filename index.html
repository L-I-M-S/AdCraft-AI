Frequently asked questions

Is AdCraft AI really a static website?

Yes. The UI is a static site you can deploy to any static host. AI calls can be done directly from the browser with a user API key, or via a lightweight backend endpoint that you control.

Can I plug in my own AI provider or model?

Absolutely. The generator form is wired so you can replace the mock generator in app.js with your own backend or direct API calls to the provider of your choice.

Which platforms does the generator support?

Out of the box, it handles Facebook, Instagram, TikTok, YouTube, LinkedIn and Google Ads formats. You can extend prompts and UI for other channels.

Does this replace human copywriters?

No. It gives them leverage. AdCraft handles structure, first drafts and volume while humans focus on insight, testing and brand nuance.
